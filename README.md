# Open App (Windows)

在 VS Code 侧边栏中快速打开 Windows 桌面应用程序。

本扩展专为 Windows 用户设计，允许您将常用的外部应用程序（如浏览器、工具软件、命令行工具等）添加到 VS Code 的侧边栏中，实现一键快速启动，无需离开编辑器。

## 功能特性

- **应用管理**：在侧边栏的 "Open App" 视图中管理您的常用应用。
- **快速启动**：点击图标即可启动应用程序。
- **添加应用**：支持手动添加 `.exe`, `.bat`, `.cmd` 等可执行文件。
- **自动导入**：一键扫描桌面和开始菜单，批量导入已安装的应用程序。
- **右键菜单**：支持编辑应用名称或从列表中移除应用。
- **持久化存储**：应用列表自动保存，重启 VS Code 后依然存在。

## 系统要求

- **操作系统**: Windows (仅支持 win32 平台)

## 使用方法

### 1. 访问视图
安装插件后，在 VS Code 左侧活动栏（Activity Bar）中找到 "Open App" 图标（通常是一个应用窗口图标）。点击即可打开应用列表视图。

### 2. 添加应用
- **手动添加**：点击视图标题栏上的 "**+**" (添加应用) 按钮，在文件选择对话框中选择您想要添加的 `.exe` 或脚本文件。
- **自动导入**：点击视图标题栏上的云下载图标 (自动导入应用)，插件会自动扫描您的桌面和开始菜单，您可以勾选需要导入的应用。

### 3. 启动应用
在列表中点击任何应用条目，即可立即启动该程序。

### 4. 管理应用
- **编辑**：右键点击应用条目，选择 "编辑应用" 修改显示的名称。
- **删除**：右键点击应用条目，选择 "删除应用" 将其从列表中移除。
- **清空**：点击视图标题栏上的清除图标，可以清空所有已添加的应用。

## 快捷键与命令

您也可以通过 `Ctrl+Shift+P` 打开命令面板，输入以下命令使用：

- `Open App: 添加应用` (open-app.addApp)
- `Open App: 自动导入应用` (open-app.importApps)
- `Open App: 刷新列表` (open-app.refreshList)
- `Open App: 清空应用` (open-app.clearApps)

## 注意事项

- 本插件目前仅支持 Windows 操作系统。
- 启动应用时实际上是调用系统的 `explorer.exe` 或直接执行文件，请确保目标文件路径正确且可访问。
